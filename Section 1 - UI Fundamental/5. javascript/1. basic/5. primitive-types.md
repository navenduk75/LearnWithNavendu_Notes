## Primitive/Value types:

```javascript
let name = 'Rahul Kumar'; //string literal
let age = 20; //number literal
let isIndian = true; //boolean literal. It can be true or false

let firstName;
let firstName = undefined; //same as before

let lastName = null; //we use null when we need to EXPLICITLY clear the value of a variable.
//Ex:
let selectedMovie = 'Sholay'; //set selectedMovie 
let selectedMovie = null; //clear selectedMovie 
let selectedMovie = 'Shaan'; //set selectedMovie again
```
